# C1TM1B - Workshop 1

## Before workshop 1

Students are expected to have watched the videos for Lecture 1 and Lecture 2 before attending Workshop 1.

To avoid spending workshop time on installation problems, we recommend you run the notebooks in Google Colab during class. You can later set up the full `homl3` environment on your own computer, following [Aurelien Geron's instructions](https://github.com/ageron/handson-ml3), if you want a more powerful local setup.

## Purpose

In this workshop, you will work in small groups on a guided machine learning mini-project. You will explore the California housing dataset, build a simple preprocessing pipeline, and train a baseline regression model. The focus is to understand each step clearly, not to build a perfect model. At the end of the workshop, each group will present one small result.

## Workshop flow

- Start with `notebooks/00_check_environment.ipynb` to verify your setup.
- Continue to `notebooks/01_exploration.ipynb` to inspect and understand the dataset.
- Finish with `notebooks/02_preprocessing_and_baseline.ipynb` to preprocess data and train a baseline model.

## Using an LLM

You are allowed and encouraged to use an LLM to help with code ideas and debugging. Run code in small steps, read errors carefully, and discuss with your group what each part of the code does before moving on.
